Teclas de Atalho (Parte 2)

Estes atalhos são ideais para gerenciar documentos e páginas da internet com mais agilidade:

Ctrl + A: Seleciona de uma só vez todo o texto ou todos os arquivos visíveis na tela.

Ctrl + F: Abre uma barra de busca rápida para localizar uma palavra específica dentro de um texto ou site.

Ctrl + S: Salva instantaneamente as alterações no arquivo que você está editando no momento.
