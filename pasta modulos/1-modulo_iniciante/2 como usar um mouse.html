Como usar o Mouse

O mouse serve para movimentar a seta na tela e apontar para o que você quer fazer. Veja como usar cada parte dele:

Botão Esquerdo (o mais usado): serve para clicar e selecionar as coisas. É com ele que você escolhe um botão ou marca um texto.

Botão Direito: ele serve para abrir um menu com opções extras que ficam escondidas. Por exemplo, se você clicar com o botão direito em cima de uma foto ou de um texto, vai aparecer uma listinha de opções do que você pode fazer com aquilo, como “Copiar”, “Colar” ou “Excluir”.

Botão Central (Scroll): gire essa rodinha para cima ou para baixo para rolar a página. Isso ajuda a ler um texto grande ou navegar por um site sem precisar arrastar as barras laterais da tela.

“Dois cliques” ou “duplo clique”: dê dois cliques rápidos com o botão esquerdo para abrir programas.

O que é um programa?

Pense no computador como se fosse uma oficina ou uma cozinha. Os programas são as ferramentas que você usa para fazer tarefas específicas.

Exemplos:

Quer escrever um texto? Você abre o programa de texto.
Quer entrar na internet? Você abre o programa de navegação.
Quer ver um vídeo? Abre o programa de vídeos.

No celular, nós chamamos os programas de “aplicativos” (ou apps). No computador, o conceito é exatamente o mesmo!
