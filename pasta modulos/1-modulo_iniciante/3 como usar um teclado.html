Como Usar o Teclado

As letras e números servem para escrever. A barra de espaço (a tecla mais longa do teclado) dá um espaço entre as palavras. A tecla Enter serve para pular de linha ou confirmar uma ação.

O Backspace (tecla com uma seta para a esquerda, acima do Enter) apaga o que foi escrito.

O Caps Lock (ou Fixa) deixa as letras maiúsculas. (Nota: na maioria dos teclados existem duas teclas “Enter” para facilitar o uso).

A tecla Shift (aquela com uma seta apontando para cima, logo abaixo do Caps Lock) serve para duas coisas:

1. Colocar uma letra maiúscula: em vez de ligar e desligar o Caps Lock, basta segurar o Shift e apertar a letra.

2. Usar os símbolos de cima: algumas teclas têm dois desenhos (como o número 1 e o ponto de exclamação !). Se você apertar a tecla normal, sai o número. Se você segurar o Shift e apertar a tecla, sai o símbolo de cima.
